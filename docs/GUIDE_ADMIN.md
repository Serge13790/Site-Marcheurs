# Guide d'Administration - Les Joyeux Marcheurs

Ce document détaille les fonctionnalités réservées aux Administrateurs et Éditeurs du site.

## 1. Accès à l'Administration
Connectez-vous avec votre compte administrateur. Cliquez sur le bouton **"Admin"** (icône bouclier) dans le menu principal ou via le menu utilisateur en haut à droite.

## 2. Tableau de Bord
La page d'accueil de l'admin vous donne une vue d'ensemble :
*   Statistiques clés (Nombre de membres, randos, photos).
*   Liste des dernières inscriptions.
*   **Actions Rapides** pour naviguer efficacement.

## 3. Gestion des Membres ("Utilisateurs")
*   **Validation** : Les nouveaux inscrits sont "En attente". Vous devez cliquer sur le bouton **✅ Valider** pour leur donner accès au site ou le révoquer en cliquant sur le bouton **❌ Refuser**.
    *   *Note : Un email automatique leur est envoyé dès validation.*
*   **Rôles** : Vous pouvez promouvoir un membre en "Éditeur" ou "Admin".

## 4. Gestion des Randonnées
*   **Création** : Créez de nouvelles fiches rando.
    *   **Statut Brouillon** : La rando est invisible pour les membres, seul l'admin reçoit une notif.
    *   **Statut Publiée** : La rando est visible et **tous les membres reçoivent automatiquement un email** pour les prévenir.
*   **Modification** : Mettez à jour les infos (heure, lieu, map GPX...).

## 5. Modération des Photos
*   Rendez-vous dans la section **Photos**.
*   Vous voyez toutes les photos téléchargées par les membres, triées par date.
*   Pour supprimer une photo inappropriée :
    1.  Survolez la photo.
    2.  Cliquez sur la **corbeille rouge**.
    3.  Confirmez la suppression dans la fenêtre qui s'ouvre.
    *   *Action irréversible : supprime la photo du site et de la base de données.*

## 6. Notifications Automatiques
Le site gère les emails pour vous :
*   **Nouvelle inscription** → Email à l'Admin pour validation du compte.
*   **Compte validé** → Email au Membre.
*   **Nouvelle Rando (Publiée)** → Email à **TOUS** les membres (Newsletter).
*   **Nouvelle Rando (Brouillon)** → Email à l'Admin.
*   **Nouvelle Photo** → Email à l'Admin (pour surveillance).
